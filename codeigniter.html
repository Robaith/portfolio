Data Center for Health Care

An web based Database management system developed with codeigniter framework .
ALhamdulillah. As a team leader , I finished my final semester project. U can read the documentation slide of this project.

  1. Patient module :
    • View appointment list and status with a doctor.
    • View the doctor list.
    • Manage own profile.
  2. Doctor module :
    • Create and manage appointments with a patient.
    • Create a prescription for the patient.
    • See patients with previously prescribed medicines.
    • Add new patient.
    • Manage own profile.
  3. Pharmacy module :
    • View the prescription of patients.
    • Provide medicine according to prescriptions.
    • Manage own profile.
